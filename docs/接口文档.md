## 1. 通用说明

### 1.1 请求规范

#### **请求头（Request Headers）**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| Content-Type | String | 是 | application/json |
| Authorization | String | 是* | Bearer {token}（除登录/注册接口外必填） |

#### **请求示例**

```http
POST /api/v1/completion HTTP/1.1
Host: localhost:7860
Content-Type: application/json
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

{
  "user_input": "你好",
  "dialog_id": "dialog_xxx"
}
```

---

### 1.2 响应规范

#### **统一响应格式**

```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "timestamp": 1715664000000
}
```

#### **响应字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| code | Integer | 状态码，0表示成功，非0表示失败 |
| message | String | 响应消息，成功返回"success"，失败返回错误信息 |
| data | Object/Array/String | 响应数据 |
| timestamp | Long | 服务器时间戳（毫秒） |

#### **流式响应格式（SSE）**

流式接口使用 Server-Sent Events (SSE) 协议：

```
data: {"type": "tool", "tool_name": "search", "tool_args": {...}, "message": "正在搜索..."}

data: {"type": "content", "content": "生成的回复内容片段"}

data: {"type": "end", "message_id": "msg_xxx"}
```

**事件类型说明：**

| type | 说明 | data字段 |
|------|------|----------|
| tool | 工具调用事件 | tool_name, tool_args, message |
| content | 内容生成事件 | content（文本片段） |
| end | 流结束事件 | message_id |
| error | 错误事件 | message |

---

### 1.3 分页规范

#### **分页请求参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | Integer | 否 | 1 | 页码 |
| limit | Integer | 否 | 10 | 每页数量 |

### 16.3 接口调用示例

#### **Python示例**

```python
import requests
import json

# 1. 用户登录
response = requests.post("http://localhost:7860/api/v1/user/login", json={
    "user_name": "zhangsan",
    "user_password": "password123"
})
token = response.json()["data"]["access_token"]

# 2. 创建对话
headers = {"Authorization": f"Bearer {token}"}
response = requests.post(
    "http://localhost:7860/api/v1/dialog",
    json={"name": "测试对话", "agent_id": "agent_xxx", "agent_type": "general"},
    headers=headers
)
dialog_id = response.json()["data"]["dialog_id"]

# 3. 发送消息（流式）
response = requests.post(
    "http://localhost:7860/api/v1/completion",
    json={"user_input": "你好", "dialog_id": dialog_id},
    headers=headers,
    stream=True
)

for line in response.iter_lines():
    if line and line.startswith(b'data: '):
        data = json.loads(line[6:])
        print(data)
```
